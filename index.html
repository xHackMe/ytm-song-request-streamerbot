🎓 Tutorial: First steps

1. Interface and Features

• Left Panel (Base): Here you will find tracks from your default playlists. The player randomly shuffles songs from them when the viewer queue is empty. You can add more playlists or remove the default one in the Playlist Manager (📁).
• Middle Panel: Bot & API status on the left, !SR switch in the center, Settings gear icon on the right, and the main YouTube player.
• Right Panel (!SR Queue): Viewer requests from Twitch. You can Drag & Drop tracks here with your mouse to easily change their order!

2. HTTP Server in Streamer.bot

For the YouTube player and API to work without blocks, the file must be run through a local HTTP server. We will use the built-in server in Streamer.bot for this.

1. In Streamer.bot go to the Servers/Clients -> HTTP Server tab.
2. Set Port: (you can leave the suggested default!)
3. In the Mappings field, click the folder icon and point to the folder where this HTML file is located.
In the PATH field, specify the subfolder name for the page:
(This will be part of the website address! e.g., 'ytm' = http://localhost:7474/ytm)
4. Check Auto Start and click Start Server.

3. WebSocket Server in Streamer.bot

For chat commands (!sr, !skip, deleting, etc.) to work, you need an active WebSocket connection.
