🔑 Google API Key Setup

🔒 Your data is 100% safe! Your API key is saved only locally in your browser (Local Storage). It is never sent to any external servers (except directly to Google to fetch tracks).
Where to get a free API key?
  1. Log in to Google Cloud Console.
  2. Create a new project (or select an existing one).
  3. In the left menu click 'APIs & Services' -> 'Library'.
  4. Search and enable: 'YouTube Data API v3'.
  5. Go to the 'Credentials' tab -> 'Create credentials' -> 'API key'.
  6. Copy the generated key and paste it in the field above.

• Left Panel (Base): Here you will find tracks from your default playlists. The player randomly shuffles songs from them when the viewer queue is empty.
